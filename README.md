## 介绍
DR 工具箱是一款可以帮助用户更加方便的使用 [梦幻框架](https://drea.cc) 的框架插件。这是它的 [主页](https://drea.cc/forum.php?mod=viewthread&tid=1028&fromuid=738)。
## 特色
- 📑多框架管理
- 📋便捷日志查看
- 🧩管理已安装插件
- 💭管理机器人所加群
- 🗳️远程命令
- 🎨更多个性化
- 🔐插件安全扫描
